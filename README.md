# plantUMLstyle
A style for plantUML

## Usage
Code example for dark theme

```
@startuml
!define DARK
!includeurl https://raw.githubusercontent.com/XaBerr/plantUMLstyle/master/style.puml
' your code here
@enduml
```

Code example for light theme

```
@startuml
!define LIGHT
!includeurl https://raw.githubusercontent.com/XaBerr/plantUMLstyle/master/style.puml
' your code here
@enduml
```
